This project focuses on predicting product sales using advertising data from TV, radio, and newspaper 
campaigns. The notebook covers preparing the dataset, building a linear regression model, and checking 
how accurately the model can estimate sales based on different forms of advertising.
Tools Used:
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
What’s Included:
Selecting and cleaning the data
Creating a correlation heatmap
Training a linear regression model
Measuring performance using MAE, RMSE, and R²
Comparing actual sales with predicted values
